Prototype for a Game Creator IDE in its early stages written in Qml.

## What you'll need
Qt-creator 2.6+
or
qmlviewer


### Run the App
If you have Qt-creator open "Game_Creator.pro" and run the app.

To run from the command line type:
    qmlviewer qml/main.qml
